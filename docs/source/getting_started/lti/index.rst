Chirun LTI Provider
===================

The Chirun LTI Provider is designed to integrate with your institution's Virutal Learning Envionment (VLE).
Before the Chirun LTI Provider can be used, it must be `setup <Chirun LTI Provider Setup>`_ on a server run
by your institution and registered with the VLE by your IT System Administrators.
